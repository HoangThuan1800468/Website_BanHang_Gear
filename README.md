# Website_BanHang_Gear
- Sử dụng mô hình MVC xử lý đường dẫn
- Chủ yếu sử dụng PHP làm ngôn ngữ backend
- Database MySql
- Sử dụng Bootrap 
* Chức năng cơ bản:
 + Trưng bày sản phẩm
 + Giỏ hàng
 + Đặt hàng
 + Quản lý sản phẩm, đơn hàng, nhân viên, tài khoản
=> Sản phẩm là bài tập lớn có sự hỗ trợ giao diện.
